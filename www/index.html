Welcome to Ferrix File Server!

This is the default index page served by Ferrix, a Rust-based network stack implementation.

You can place files in the www directory to serve them through this HTTP server.

Features

  • Custom TCP/IP stack implementation
  • HTTP file server with path-based routing
  • TUN device integration
  • Basic MIME type detection

Try these paths:

  • /test.txt - Plain text file
  • /about.html - About page
